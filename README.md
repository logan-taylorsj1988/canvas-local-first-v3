## What Canvas is for

Canvas is made for people who want ideas, sketches, and notes to stay right next to the work itself. Instead of splitting writing, visual thinking, and planning across separate tools, it brings them into one macOS desktop workspace built around an open-ended canvas.

Its persistence model is file-based and local, so the workspace remains stored on your machine in a way that is straightforward to manage. With folder organization, unlimited pages, and a freeform layout, Canvas works well for writers, makers, and creative teams assembling drafts, concepts, or project plans.

- Infinite canvas for open-ended notes, sketches, and arrangement

## How to use Canvas

Canvas is a workspace for collecting ideas before they become posts, outlines, or formal documents.

Typical ways to work with it include:

- Make one page per project, draft, or content stream
- Place notes, sketches, links, and images directly on the canvas
- Move related items into folders as the project expands
- Capture a carousel view when you need a cleaner presentation layout
- Export a page or JPEG when it is time to share or publish

For planning content, many users keep research, rough writing, and visual references together so the draft can develop in one place without constant tool switching.

## Configuration

Canvas keeps your workspace local, so setup mostly comes down to deciding where the vault should live on your Mac.

Common ways to organize it:

- Pick a dedicated local folder for the vault
- Separate projects into different folders for easier access later
- Use page-level exports when you want to save a draft snapshot

If your build includes preferences, store them with the local workspace so the structure remains easy to back up and move.

## FAQ

**How is Canvas different from a cloud-first note app?**  
Canvas is built around local file-based storage, so your workspace stays on disk instead of being centered on remote syncing.

**Can I use it for writing and drawing together?**  
Yes. The infinite canvas is intended to hold text, sketches, images, and links together.

**Where are my files stored?**  
Canvas uses a local vault, which keeps workspace data in a file-based structure on your machine.

**How do I share a page or layout?**  
Use page export for a page snapshot or JPEG export for a visual capture.

**What should I do if I have trouble opening the app?**  
Make sure you downloaded the right macOS build, verify file permissions for the vault location, and confirm that enough local storage is available.
